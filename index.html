תוסף הנגישות של ליאור — דמו

איך זה מרגיש — הדגמה

לחיצה על הכפתור בתחתית העמוד (או על "פתח תפריט נגישות") תפתח את ווידג'ט ההטמעה.

סקירה קצרה

  • מוכן לשימוש — ללא תלות חיצונית.
  • נבדק לפי WCAG 2.1 AA.
  • ניתן לשמור פרופילים ולהגדיר מפתחות לקוח.

איך לשלב — קוד

העתק/י והדבק/י בתוך <head> או לפני סגירת <body> (הספרייה נטענת עם defer):

<!-- העתק את הקובץ שהעלית: widget-v2.js -->
<script src="widget-v2.js"
        data-client-key="TEST-555"
        defer></script>

דוגמה: ניתן לשנות data-client-key למפתח של הלקוח. ללא מפתח תקין הווידג'ט יחזור ויחסום את עצמו (ראה LICENSE_MANAGER בקוד).

הפעלה מהעמוד (JS)

כדי לפתוח את הפאנל דרך כפתור — השתמש בקוד הבא:

document.getElementById('open-widget').addEventListener('click', e => {
  e.preventDefault();
  // אם הווידג'ט יוצר אלמנט עם id = lior-acc-root, נחפש ונלחץ על הכפתור הפנימי:
  const root = document.getElementById('lior-acc-root');
  if (root) {
    const btn = root.querySelector('.lior-acc-button');
    if (btn) btn.click();
  } else {
    alert('הווידג\'ט טרם נטען. רענן את הדף או הטעין את התסריט עם defer.');
  }
});

הצהרת נגישות קצרה

התוסף נועד לשפר נגישות עבור משתמשי קצה — אך חשוב לבצע בדיקות משלימות (NVDA/VoiceOver/JAWS) ולספק טקסט אלטרנטיבי לתמונות, כותרות הגיוניות וניווט לוגי.

טיפ פרקטי: אל תסתמכו רק על ווידג'ט — בצעו בדיקות ידניות ואוטומטיות.
