ופשוטה להטמעה

פתרון הנגישות המתקדם של ZeepClick. ללא שרתים, ללא עלויות נסתרות, ועם התאמה מלאה לצרכי המשתמש.

פתח תפריט נגישות הוראות התקנה

יכולות ליבה

פרופילים חכמים

התאמה מיידית לדיסלקסיה, לקויי ראייה וצרכים קוגניטיביים.

מנוע טקסט

שליטה מלאה בגודל, ריווח וגופן קריא בזמן אמת.

הקראה קולית

מנוע TTS (Text-to-Speech) להקראת תוכן האתר.

מצב רגוע

נטרול הבהובים ואנימציות למניעת עומס חושי.

  • ללא תלות חיצונית — הקוד רץ בצד הלקוח בלבד
  • תאימות WCAG 2.1 AA — נבנה לפי הסטנדרטים המחמירים
  • ביצועים מעולים — השפעה אפסית על זמן טעינת האתר
  • פרטיות מובנית — המידע נשמר בדפדפן המשתמש בלבד

הצהרת נגישות

התוסף של ZeepClick מסייע בהנגשת האתר, אך אינו מחליף הנגשה בקוד המקור. אנו ממליצים על שילוב של התוסף יחד עם כתיבת קוד סמנטי ותקני.

מומלץ לבצע בדיקות תקופתיות עם קוראי מסך (כגון NVDA או VoiceOver).

הטמעה מהירה

העתק את הקוד הבא לפני סגירת תג </body>:

<!-- ZeepClick Widget v2.0 -->
<script
  src="https://liorsal.github.io/negi-vs/widget.min.js"
  data-client-key="YOUR-KEY"
  defer>
</script>

יש להחליף את YOUR-KEY במפתח הלקוח שקיבלת.

שליטה למפתחים

const root = document.getElementById('lior-acc-root');
if (root) {
  const btn = root.querySelector('.lior-acc-button');
  if (btn) btn.click();
}
