תוסף נגישות

פתרון נגישות מתקדם עם מצבים מוכנים מראש, התאמות אישיות וקריאה בקול — מוכן להטמעה מיידית

פתח תפריט נגישות

מה התוסף עושה

A

מצבים מוכנים מראש

פרופילים מותאמים לדיסלקסיה, לקויי ראייה וצרכים קוגניטיביים

T

התאמות טקסט

הגדלת טקסט, ריווח אותיות ומילים, וביטול אנימציות

R

קריאה בקול

Text-to-Speech מובנה לדפדפנים התומכים

איך זה עובד

לחץ על כפתור הנגישות בתחתית העמוד או על הכפתור למעלה כדי לפתוח את תפריט ההגדרות ולהתאים את האתר לצרכים שלך.

יתרונות

  • מוכן לשימוש — ללא תלות חיצונית
  • תואם WCAG 2.1 AA
  • שמירת הגדרות אוטומטית
  • ניהול מפתחות לקוח

איך לשלב

העתק והדבק את הקוד הבא לפני סגירת תג </body>:

<!-- Lior Accessibility Widget v2.0 --> <script src="https://liorsal.github.io/negi-vs/widget.min.js" data-client-key="ROEE-TATTOO-01" defer> </script>

ניתן לשנות את data-client-key למפתח הלקוח שלך.

הפעלה מתוכנית

כדי לפתוח את הפאנל דרך JavaScript:

const root = document.getElementById('lior-acc-root'); if (root) { const btn = root.querySelector('.lior-acc-button'); if (btn) btn.click(); }

הצהרת נגישות

התוסף נועד לשפר את הנגישות עבור משתמשי קצה, אך חשוב לבצע בדיקות משלימות עם כלי נגישות (NVDA, VoiceOver, JAWS) ולספק תמיכה מלאה בתקני WCAG.

טיפ: אל תסתמכו רק על ווידג'ט — בצעו בדיקות ידניות ואוטומטיות באופן קבוע.
