תוסף נגישות מתקדם
לאתר שלך

פתרון נגישות מלא עם מצבים מוכנים מראש, התאמות אישיות וקריאה בקול. מוכן להטמעה מיידית, ללא תלות בשרתים חיצוניים.

פתח תפריט נגישות הוראות התקנה

מה התוסף עושה

מצבים מוכנים

פרופילים לדיסלקסיה, לקויי ראייה וצרכים קוגניטיביים.

התאמות טקסט

שינוי גודל, ריווח וגופן קריא בזמן אמת.

קריאה בקול

מנוע TTS מובנה להקראת תוכן האתר.

ביטול הבהובים

עצירת אנימציות למניעת עומס חושי.

יתרונות מרכזיים

  • עצמאי לחלוטין — אין תלות בשרתים חיצוניים או שירותי צד ג'
  • תואם תקנים — נבנה בהתאם להנחיות WCAG 2.1 רמה AA
  • קל משקל — טעינה מהירה ללא פגיעה בביצועי האתר
  • שמירה מקומית — זוכר את העדפות המשתמש לביקורים חוזרים

הצהרת נגישות

התוסף נועד לשפר את הנגישות עבור משתמשי קצה ולסייע בעמידה בתקנות. עם זאת, התקנת תוסף אינה מחליפה את הצורך בהנגשת קוד המקור של האתר.

מומלץ לבצע בדיקות תקופתיות עם קוראי מסך (כגון NVDA או VoiceOver) ולוודא שהאתר עצמו בנוי בצורה סמנטית ונגישה.

איך לשלב

העתק והדבק את הקוד הבא לפני סגירת תג </body>:

<!-- Lior Accessibility Widget v2.0 -->
<script
  src="https://liorsal.github.io/negi-vs/widget.min.js"
  data-client-key="YOUR-KEY"
  defer>
</script>

החלף את YOUR-KEY במפתח הלקוח שלך.

שליטה דרך קוד

ניתן לפתוח את התפריט באמצעות JavaScript:

const root = document.getElementById('lior-acc-root');
if (root) {
  const btn = root.querySelector('.lior-acc-button');
  if (btn) btn.click();
}
