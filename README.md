# אתר נגישות - כפתור נגישות

אתר מודרני עם כפתור נגישות מתקדם המאפשר התאמה אישית של חוויית הגלישה.

## תכונות

### כפתור נגישות צף
- כפתור נגישות קבוע בפינה השמאלית העליונה
- פתיחה וסגירה של פאנל הגדרות נגישות
- תמיכה מלאה בניווט במקלדת

### הגדרות נגישות

1. **גודל טקסט**
   - הגדלה והקטנה של הטקסט באתר
   - טווח: 75% עד 200%
   - עדכון מיידי

2. **מצב ניגודיות גבוהה**
   - הפעלת מצב ניגודיות גבוהה לשיפור הקריאות
   - צבעים מותאמים לנגישות

3. **סמן גדול**
   - סמן גדול יותר לנוחות השימוש

4. **קו תחתון לקישורים**
   - הוספת קו תחתון לכל הקישורים באתר

5. **גופן קריא**
   - החלפה לגופן קריא יותר

6. **איפוס הגדרות**
   - איפוס כל ההגדרות לערכי ברירת מחדל

## תכונות טכניות

- **שמירת הגדרות**: ההגדרות נשמרות ב-localStorage ונשארות גם אחרי רענון הדף
- **תמיכה ב-ARIA**: תמיכה מלאה בתקני נגישות ARIA
- **תגובתיות**: האתר מותאם למובייל ולטאבלט
- **ניווט במקלדת**: תמיכה מלאה בניווט במקלדת
- **תמיכה בעברית**: תמיכה מלאה בעברית עם RTL

## שימוש

1. לחצו על כפתור הנגישות בפינה השמאלית העליונה
2. בחרו את ההגדרות הרצויות
3. ההגדרות יישמרו אוטומטית

## טכנולוגיות

- HTML5
- CSS3 (עם משתנים CSS)
- JavaScript (ES6+)
- תמיכה ב-localStorage

## דפדפנים נתמכים

- Chrome (מומלץ)
- Firefox
- Safari
- Edge

## התקנה

1. שכפלו את הפרויקט:
```bash
git clone git@github.com:liorsal/ng-w.git
```

2. פתחו את `index.html` בדפדפן

או הפעילו שרת מקומי:
```bash
python -m http.server 8000
```

ואז פתחו: `http://localhost:8000`
